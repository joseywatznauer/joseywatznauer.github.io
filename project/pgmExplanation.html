Recursive Mapping Fibonacci Sequence Python Program

In mathematics, the Fibonacci Sequence is a sequence in which each element is the sum of the two elements that precede it. Originating from Italian mathematician Fibonacci's 1202 book Liber Abaci, the original problem concerned the reproduction of rabbits under perfect circumstances, and today, the "Golden Ratio" that the sequence produces is known to appear often in nature.
The Fibonacci sequence is as follows:

Fn = Fn-1 + Fn-2

The following is a Python program that calculates the requested Fibonacci sequence value:

        fibMap = {}

        def fib(n):
            if n < 3: #because first two values of the sequence are 1
                return 1
            elif n in fibMap.keys():
                return fibMap[n]
            else:
                result = fib(n-1) + fib(n - 2)
                fibMap[n] = result
                return result
            
        inputNum = int(input("Which value of the Fibonacci sequence would you like? "))
        answer = fib(inputNum)
        print("Answer: ", str(answer))

The program prompts the user for whichever value in the sequence they would like to see. If the user were to request the 6th value, he or she would enter 6, and this value would be assigned to inputNum. When the recursive function reaches the base case, the program would output Answer: 8, because 8 is the 6th value of the Fibonacci sequence.
This program, known as RMFSPP, finds the Fibonacci sequence value using recursion. Recursion refers to a function that calls itself. This specific recursive program stores the known values in a dictionary so each time the "previous" value is needed to find the next value, it can find it in the dictionary instead of redoing all of the previous calcutions. This saves exponential amounts of time and processing power.
